# N8N_MASTER_DEPLOYMENT_PACKAGE

This folder is a **single consolidated drop-in package** of n8n workflows + docker stack + schemas + docs.

Start here:
- `00_DOCS/QUICK_START.md`
- `00_DOCS/WORKFLOWS_INDEX.md`
- `00_DOCS/IMPLEMENTATION_ORDER.md`

If you want “just import everything”:
- import all JSON under `03_WORKFLOWS/`
- then set workflow IDs in `.env` per `00_DOCS/WORKFLOW_ID_MAPPING.md`
